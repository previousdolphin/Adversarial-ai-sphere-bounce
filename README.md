# Adversarial-ai-sphere-bounce

A highly interactive, media-led landing page designed for by Jesse for Jesse. It features a custom vanilla JavaScript engine that securely preloads video assets, presenting the user with a loading screen before revealing a central interactive image floating in a white void. 

Clicking the image triggers seamless video playback that finishes with a custom programmatic "rubberband" effect, playing the video in accelerated reverse back to the original still frame.
